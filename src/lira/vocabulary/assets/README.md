# Vocabulary Layer Assets

Generated and reference data consumed by the Vocabulary Layer at
seeding time -- not source code, not the authoritative record of
anything. See `common/en/README.md` for the English Common Vocabulary
Cache.

`example_ui/` is different in kind from `common/`: it is a static,
pre-generated example of `DictionaryView`'s HTML output, not something
any seeder loads at runtime. See `example_ui/README.md`.
